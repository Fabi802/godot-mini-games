Tic Tac Toe
A simple **Tic Tac Toe game** built with the **Godot Engine**, featuring a local multiplayer mode.  
This project was created to practice game logic, UI design, and turn-based mechanics.

Gameplay:
- play the classic Tic Tac Toe game on a 3x3 grid
- Two players take turns: ❌ and ⭕
- First player to get 3 in a row wins
- If the grid is full with no winner, the game ends in a draw
- Restart the game anytime


Multiplayer:
- This game supports local multiplayer or playing against a bot:
- Player 1 vs Player 2 on the same device
- Simple turn-based system


Features:
- Local 2-player multiplayer mode
- Turn-based gameplay system
- Win and draw detection
- Simple and responsive UI
- Restart / reset functionality
- Clean grid-based logic


Built With:
- Godot Engine  
- GDScript


What I Learned:
- Grid-based logic implementation
- Win condition checking algorithms
- UI updates based on game state
- Handling user input in a structured way
- Building small complete game systems
