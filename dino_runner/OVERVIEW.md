Dino Runner

A 2D endless runner inspired by the classic Chrome Dinosaur Game, developed with the **Godot Engine**. This project focuses on procedural obstacle spawning, increasing difficulty, and responsive player controls.


Gameplay:
Run as far as possible while avoiding obstacles.
- Press Space  to jump
- Dodge incoming obstacles
- Survive as long as possible to achieve a new high score


Features:
- Endless runner gameplay
- Procedurally spawned obstacles
- Score and persistent high score system
- Smooth player movement and jumping
- Game over and restart functionality
- Clean and responsive UI



Built With

- Godot Engine
- GDScript


What I Learned
- Game loops and endless gameplay mechanics
- Procedural obstacle generation
- Collision detection
