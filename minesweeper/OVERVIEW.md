Minesweeper Clone

A classic Minesweeper game recreated using the Godot Engine.
This project was built to practice grid-based game logic, UI systems, and algorithmic thinking such as recursive tile revealing.

Gameplay

The goal is to clear the board without triggering any mines.

Left-click to reveal a tile
Right-click to place or remove a flag
Numbers indicate how many mines are adjacent
Clear all safe tiles to win


Features
Classic Minesweeper mechanics
Random mine generation each game
Grid-based system
Flag system for marking suspected mines
Win and game-over detection
Restart functionality
Clean and simple UI

Built With
Godot Engine
GDScript

What I Learned
Grid-based data structures
Game state management
User input handling (mouse interaction)
UI feedback systems
Randomized level generation
